## 🗂️ Web Based Task Allocation System

A web application to assign tasks to IT employees based on 
their skills and workload.

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python (Flask)
- **Database:** MySQL

## ✨ Features
- 👤 Role-based login (Admin, Manager, Employee)
- 📋 Task creation and assignment based on skills
- 📊 Workload monitoring and management
- 🔔 Notification system
- 📝 Task status tracking
- 📜 Allocation history log
